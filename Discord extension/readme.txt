Download deps folder https://drive.google.com/file/d/17HsgCKSq3sQRqS4b8mbsgp609SgX3yLe/view?usp=sharing

Place deps folder in root of directory. Compile in x86, x64 will not compile.

To get the Discord channel id, right click on the channel > Copy ID. (in dev mode)

Visit https://discordapp.com/developers/applications/me and make a user bot and get the copy token.

Edit the settings.ini file and add your bot token and channel ID.
